## Build Instructions
### Maven Build
- Build Commands:
    - Clean build: `mvn clean`
    - Compile project: `mvn compile`
    - Package project: `mvn package`
    - Install to local repository: `mvn install`
    - Deploy project: `mvn deploy`
- Build Process:  
    The Maven build process mainly includes the following phases:
    1. Clean (clean): Deletes files generated by previous builds
    2. Compile (compile): Compiles project source code
    3. Test (test): Runs unit tests
    4. Package (package): Packages compiled code into a distributable format (e.g., JAR, WAR)
    5. Verify (verify): Runs checks to validate the package
    6. Install (install): Installs the package into the local repository
    7. Deploy (deploy): Copies the final package to a remote repository
- Package Directory:  
    - Default package directory is `target/`
    - Compiled class files are located in `target/classes/`
    - Test class files are located in `target/test-classes/`
    - The final package file (e.g., JAR) is located in the `target/` directory
    - For this project, the packaged file will be `target/trie-1.0-SNAPSHOT.jar`
